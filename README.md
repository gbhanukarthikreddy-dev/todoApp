# 📋 Todo App

A modern and minimal Todo application built with **React Native**, **Expo SDK 54**, **NativeWind**, and **AsyncStorage**. Easily manage your daily tasks with a clean and responsive user interface.

---

## ✨ Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed
- 💾 Persistent storage using AsyncStorage
- 📱 Responsive UI
- 🎨 Styled with NativeWind (Tailwind CSS for React Native)
- ⚡ Fast and lightweight

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/todo-app.git
```

Navigate to the project

```bash
cd todo-app
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npx expo start
```

Run on Android

```bash
npx expo run:android
```

Run on iOS

```bash
npx expo run:ios
```
